Setup Guide
Step-by-step flow to pair and test a device
1
Start ZeroClaw

Run zeroclaw gateway with the hub URL in config, then run zeroclaw register in another terminal to create a pairing code.

2
Confirm pairing

Go to the Pairing page to see pending codes. In the ZeroClaw terminal running register, enter the code when prompted.

3
Device connects

After pairing, the device appears on the Gateway page. Click Use to see all tools available on that device.

4
Use any tool

Pick a tool from the grid. A dynamic form appears with all parameters. Fill in what you need, click Run, and see the result. For cursor_cli agent tasks, status is polled automatically.

5
Advanced: MCP Console

Use the MCP Console to send raw JSON-RPC requests: initialize, tools/list, tools/call.
